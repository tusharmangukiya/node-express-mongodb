# Node

### To install dependencies
```sh
$ npm install
```

### To start server
```sh
$ npm run dev
```

### To run migration
To pre seed the roles information and admin record, you can run following command on your local machine.
```sh
$ npm run migrate:status
$ npm run migrate:up
```
